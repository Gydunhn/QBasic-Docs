    Instalación y Configuración

    QBasic es un software antiguo diseñado para MS-DOS, por lo que instalar y ejecutarlo en sistemas operativos modernos requiere el uso de emuladores o herramientas específicas.

    Obtener QBasic

    QBasic venía incluido en MS-DOS 5.0 hasta MS-DOS 6.22, así como en algunas versiones tempranas de Windows. Puedes encontrar copias archivadas en:

    • WinWorld PC - QBasic 1.x
    • Archive.org - Microsoft QuickBasic

    Los archivos principales que necesitas son:

    • QBASIC.EXE - El entorno de desarrollo integrado
    • QBASIC.HLP - El archivo de ayuda

    Instalación en sistemas modernos

    Usando DOSBox

    DOSBox es un emulador de DOS que te permite ejecutar QBasic en sistemas operativos modernos.

    1. Instala DOSBox:
    2. Windows: Descarga el instalador desde dosbox.com
    3. macOS: brew install dosbox con Homebrew

    4. Linux: sudo apt install dosbox (Ubuntu/Debian) o equivalente

    5. Configura DOSBox:

    6. Crea una carpeta en tu sistema para los archivos de QBasic (por ejemplo, C:\QBasic en Windows)
    7. Copia QBASIC.EXE y QBASIC.HLP a esta carpeta
    8. Abre el archivo de configuración de DOSBox (normalmente en ~/.dosbox/dosbox-0.74.conf en Linux/macOS o C:\Users\[Usuario]\AppData\Local\DOSBox\dosbox-0.74.conf en Windows)

    9. Añade al final:

      text [autoexec] mount c C:\QBasic c:

      (Ajusta la ruta según la ubicación de tu carpeta)

    10. Ejecuta QBasic:

    11. Inicia DOSBox
    12. Ejecuta QBASIC.EXE escribiendo qbasic y presionando Enter

    Alternativa: QB64

    QB64 es una versión moderna de QBasic compatible con sistemas actuales.

    1. Descarga QB64:

    2. Visita https://qb64.com/ y descarga la versión para tu sistema operativo

    3. Instalación:

    4. Windows: Descomprime el archivo descargado y ejecuta qb64.exe
    5. macOS/Linux: Descomprime el archivo y ejecuta ./qb64

    QB64 ofrece compatibilidad con programas QBasic originales pero añade características modernas como soporte para resoluciones más altas, más memoria, y la capacidad de compilar a ejecutables nativos.

    Configuración del entorno de desarrollo

    En QBasic original (vía DOSBox)

    1. Ajustes de pantalla:
    2. Presiona Alt+Enter para cambiar entre pantalla completa y ventana

    3. Ajusta la resolución de DOSBox en el archivo de configuración si es necesario

    4. Opciones de QBasic:

    5. Dentro de QBasic, accede al menú Options para configurar:
      • Display...: Cambia colores y disposición del editor
      • Environment...: Configura tabulaciones y otros aspectos

    En QB64

    1. Opciones de visualización:

    2. Menú View > Options para ajustar colores, fuentes y comportamiento del editor

    3. Carpetas de trabajo:

    4. Configura las carpetas por defecto desde el menú File > Preferences
